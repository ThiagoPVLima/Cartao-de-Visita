## 📸 Preview

> Interface inspirada nos clássicos **Windows 98** e **Windows XP**, com sistema de janelas arrastáveis, troca de temas, wallpapers originais e formulário de contato funcional.

---

## ✨ Funcionalidades

| Recurso | Descrição |
|---|---|
| 🪟 **Dois Temas** | Alterne entre Windows 98 e Windows XP via menu de contexto |
| 🖼️ **Wallpapers reais** | 5 wallpapers originais do Windows XP (Bliss, Azul, Autumn, Vortec Space, Red Moon Desert) |
| 🖱️ **Janelas arrastáveis** | Arraste qualquer janela pela barra de título |
| 📋 **Abas dinâmicas** | Home, Skills e Contato com animação de transição |
| 📨 **Formulário funcional** | Envio de mensagens via [Formspree](https://formspree.io) direto para o e-mail |
| 🔔 **Notificações** | Sistema de pop-ups estilo Windows com sons 8-bit gerados via Web Audio API |
| ⌨️ **Easter Egg** | `Ctrl + Alt + Del` abre o Gerenciador de Tarefas |
| 🕐 **Relógio ao vivo** | Atualiza a hora na taskbar em tempo real |
| 🎂 **Idade automática** | Calcula a idade automaticamente a partir da data de nascimento |

## 🎨 Temas

### Windows 98
Interface com visual flat estilo **Luna clássico** usando [98.css](https://jdan.github.io/98.css/). Fundo teal com padrão de grade diagonal.

### Windows XP
Visual **Luna Blue** com gradientes, bordas arredondadas e a barra de tarefas azul característica. Usa [XP.css](https://unpkg.com/xp.css).

> Troca de tema via **botão direito** no desktop → seção **Tema**.

---

## 🖼️ Wallpapers

Os wallpapers são os originais do **Windows XP**, carregados via URL direta:

- 🌄 **Bliss** — as colinas verdes icônicas
- 🌀 **Azul** — espirais azuis
- 🍂 **Autumn** — folhas de outono
- 🌌 **Vortec Space** — nebulosa espacial
- 🌙 **Red Moon Desert** — deserto com lua vermelha

> Troca via **botão direito** no desktop → seção **Plano de Fundo**.

---

## 🚀 Como rodar

Não precisa de build, servidor ou dependências. Basta abrir:

```bash
# Clone o repositório
git clone https://github.com/ThiagoPVLima/Cartao-de-Visita.git

# Abra o arquivo no navegador
open index.html
```

> ⚠️ O formulário de contato usa [Formspree](https://formspree.io) e requer conexão com a internet para envio.
